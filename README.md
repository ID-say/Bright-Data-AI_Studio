## 简要实现过程

整个过程其实很简单：

1. 打开抖音热榜页面
2. 使用 AI Studio 的 Web Scraper
3. 选择需要的字段（标题 / 热度 / 排名）
4. 导出为 CSV / JSON / ...

全程**不需要写代码**，也**不需要自己分析**网页结构
